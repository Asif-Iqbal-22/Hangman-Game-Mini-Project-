# Hangman Game in C

A simple console-based Hangman game built using C programming language.

## Features
- Randomly select word
- User guesses letters
- Show partially guessed word
- Track wrong attempts
- End game on win or lose

## How to Run
1. Compile the program:

    gcc hangman.c -o hangman

2. Run:

    ./hangman
